# Lamborghini V10 Trading Desk

Mobile-first web app in tema dark blue Lamborghini per gestire un trading desk operativo: command center, portfolio modificabile, radar opportunità, quick analyze, regime macro, rotazione settoriale, relative strength leaders, journal, graveyard, alert e settings.

## Avvio

```bash
npm install
npm run start
```

## Build

```bash
npm run build
```

## Funzioni principali

- **Opportunity Radar**: genera automaticamente opportunità con rating 1-10, conviction, trade duration, entry, stop, target e motivo operativo.
- **Quick Analyze**: inserisci un ticker e ottieni analisi completa con trend, conviction, livelli e piano rischio.
- **Portfolio**: capitale libero e posizioni modificabili, con salvataggio locale nel browser.
- **Journal / Graveyard / Alert Center**: log operativo, lezioni dagli errori e alert armabili.
